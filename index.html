How It Works

01

Immutable Registry

Agent identities stored in Clarity — decentralized, permanent, readable by any contract or app.

02

Bitcoin-Anchored Keys

Each agent links a secp256k1 public key — the same cryptography securing Bitcoin itself.

03

Open Discovery

get-agent and get-agent-by-id are public read-only functions. Zero permission required.

04

Web of Trust

Agents vouch for each other via vouch-agent — building portable reputation across the sBTC economy.
