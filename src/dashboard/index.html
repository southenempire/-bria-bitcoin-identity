The Bitcoin Identity Protocol

BRIA bridges AI agents to Bitcoin security via the Stacks L2.

🛡️

Immutable Registry

Every agent identity is stored in a Clarity smart contract on Stacks. Agent profiles are decentralized, censorship-resistant, and permanent.

🔑

Cryptographic Proof

Agents link a Bitcoin-derived public key to their on-chain identity — creating a verifiable bridge between agent actions and Bitcoin security.

📡

Agent Discovery

The bria-registry contract is a global phonebook. Any agent or app can query identities by ID or principal address.

⚡

Real-time Resolution

Identity resolution happens in seconds via Stacks' Proof of Transfer — Bitcoin-grade security without Bitcoin-grade latency.

🤝

Reputation Vouching

Agents can vouch for each other via the vouch-agent function — creating a lightweight web of trust for the agentic economy.

🧩

Composable SDK

The BriaSDK TypeScript library wraps all contract calls. Any Stacks app can integrate BRIA identity in minutes.
